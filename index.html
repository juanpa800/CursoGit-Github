Haciendo pruebas para conocer git y github

Este trabajo se está haciendo con fines educativos propios, los cuales me permitirán aprender git y github

Comandos vistos y aprendidos:

  1. git init

  2. git status

  3. git status -s

  4. git config global user.username "name"

  5. git config global user.email "email"

  6. git config -l

  7. git add <file>

  8. git add .

  9. git commit -m "mensaje del commit"

  10. git log --oneline

  11. git reset --head <ID del commit>

  12. git commit --amend

  13. git push origin master

  14. git pull

  15. git tag <nomre del tag> -m "mensaje"

  16. git push --tags

  17. git clone <URL>

  18. git branch <nombre>

  19. git branch

  20. git checkout <nombre rama>
